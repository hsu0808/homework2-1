#語法:
冪次運算：輸入num^num  例：2^2

根號運算：輸入num ** num 例：4 ** 2

for-loop運算: 輸入 for 變數 loop 起始值 終值 add 執行連續加法運算
```diff
calc > for i loop 1 10 add
calc > i
55
``` 
for-loop運算: 輸入 for 變數 loop 起始值 終值 avg 執行連續加法平均運算
```diff
calc > for i loop 1 10 avg
calc > i
5.5
```
